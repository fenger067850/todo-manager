# 粉色主题颜色配置说明

## 概述
已将整个应用的配色方案从蓝色系调整为适合女生的粉色/紫色系。

## 主要修改文件

### 1. Tailwind 配置 (`tailwind.config.ts`)
- **主色调 (primary)**：从蓝色 (#3b82f6) 改为粉色 (#ec4899)
- **四象限颜色**：
  - 紧急且重要：粉色 (#ec4899)
  - 不紧急但重要：紫色 (#a855f7)
  - 紧急但不重要：浅粉色 (#f472b6)
  - 不紧急且不重要：淡紫色 (#d8b4fe)

### 2. 全局样式 (`src/app/globals.css`)
- **背景色**：淡粉色背景 (HSL: 326 100% 99%)
- **文字色**：深粉色文字 (HSL: 326 78% 10%)
- **滚动条**：
  - 轨道：#fce7f3 (淡粉色)
  - 滑块：#f9a8d4 (粉色)
  - 滑块悬停：#f472b6 (深粉色)

### 3. 主要页面颜色

#### 首页 (`src/app/page.tsx`)
- 背景渐变：从粉色 (pink-50) 到紫色 (purple-100)
- 标题：深粉色 (pink-900)
- 文字：粉色 (pink-700)
- 登录按钮：粉色 (pink-600)
- 注册按钮：紫色 (purple-600)
- 特性卡片边框：粉色/紫色交替

#### 导航栏 (`src/components/navigation.tsx`)
- 背景：粉色到紫色渐变
- 边框：粉色 (pink-200)
- Logo：粉色 (pink-600)
- 激活状态：深粉色 (pink-900)
- 悬停状态：粉色 (pink-800)

#### 按钮组件 (`src/components/ui/button.tsx`)
- 默认按钮：粉色背景 (pink-600)，悬停 (pink-700)
- 链接样式：粉色文字 (pink-600)
- 焦点环：粉色 (pink-500)

### 4. 批量替换规则

使用 sed 命令在所有 `.tsx` 文件中执行以下替换：

```bash
# 背景色
bg-blue-* → bg-pink-*

# 文字色
text-blue-* → text-pink-*

# 边框色
border-blue-* → border-pink-*

# 悬停状态
hover:bg-blue-* → hover:bg-pink-*
hover:text-blue-* → hover:text-pink-*
hover:border-blue-* → hover:border-pink-*

# 焦点状态
focus:ring-blue-* → focus:ring-pink-*
focus:border-blue-* → focus:border-pink-*
```

## 额外的紫色点缀

为了增加视觉层次感，在某些地方使用紫色作为辅助色：
- 注册按钮：紫色 (purple-600)
- 部分特性卡片：紫色边框和文字
- 渐变背景：粉色到紫色的过渡

## 注意事项

1. **一致性**：所有交互元素（按钮、链接、输入框）统一使用粉色系
2. **可访问性**：保持了足够的对比度，确保文字清晰可读
3. **层次感**：通过不同深度的粉色/紫色营造视觉层次
4. **女性化设计**：柔和的粉色调配合圆润的边角，打造温馨友好的界面
